Ответ 3

Для того, чтобы избавиться от лишних запятых нужно использовать join,
тогда в productList будет передаваться строка из map,
а не запятые как в случае с передачей из функции renderProduct со стрелочной функцией.
const productsList = list.map(item => renderProduct(item)).join('');
